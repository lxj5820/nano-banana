# Nano-banana 图片生成器

一个基于 AI 的在线图片生成工具，支持参考图、多模型选择和批量生成。

![Preview](https://img.shields.io/badge/Platform-Web-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 功能特性

### 图片生成
- 支持多个 AI 模型：`nano-banana`、`nano-banana-2`、`nano-banana-pro`
- 可调节生成数量（1-4张）
- 多种画幅比例可选（1:1、16:9、9:16、4:3、3:4）
- 多任务并发生成，无需等待

### 参考图
- 支持拖拽或点击上传参考图
- 支持 Ctrl+V 快捷粘贴图片
- 参考图左上角显示序号标记
- 可同时添加多张参考图

### 历史记录
- 生成后自动保存到本地 IndexedDB
- 支持放大查看、下载、复用
- 批量下载为 ZIP 包
- 一键清空历史

### 用户体验
- 生成状态实时显示（占位图 → 加载中 → 完成/失败）
- 响应式设计，适配各种屏幕
- 自定义弹窗提示
- 引导指南（首次使用自动弹出）

## 快速开始

### 配置 API Key

1. 点击右上角「设置」按钮
2. 输入你的 API Key
3. 点击保存

### 生成图片

1. 在文本框输入提示词
2. （可选）添加参考图
3. 选择模型和生成数量
4. 点击「生成」按钮

### 快捷操作

| 操作 | 方式 |
|------|------|
| 粘贴图片 | Ctrl+V |
| 放大查看 | 点击放大按钮 |
| 下载单张 | 点击下载按钮 |
| 批量下载 | 历史区「下载」按钮 |
| 复用设置 | 点击复用按钮 |

## 技术栈

- **前端**：HTML5、CSS3、JavaScript
- **存储**：IndexedDB（本地历史记录）
- **AI 接口**：NewAPI (Gemini)
- **压缩**：JSZip

## 使用说明

### 模型选择

| 模型 | 特点 |
|------|------|
| nano-banana | 基础模型，响应快速 |
| nano-banana-2 ⭐ | 推荐，支持更高分辨率 |
| nano-banana-pro | 专业版，最佳效果 |

### 提示词技巧

**推荐结构**：
```
主体（数量/特征）+ 动作 + 场景（地点/时间/天气）+ 风格/媒介 + 构图/相机 + 光影/色彩 + 关键细节 + 约束条件
```

**编辑图片时**：
```
保持（不可改变）+ 改变（修改什么）+ 如何（风格/强度/方向）+ 约束条件
```
