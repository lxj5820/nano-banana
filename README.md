# 词元AI生图工坊

基于 AI 的在线图片生成工具，支持文生图、图生图、合集管理与批量下载。

![Platform](https://img.shields.io/badge/Platform-Web-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 功能特性

### 图片生成
- 多模型支持：`nano-banana`、`nano-banana-2`、`nano-banana-pro`
- 可调节生成数量（1-4张）
- 多种画幅比例（1:1、16:9、9:16、4:3、3:4、2:3、3:2、4:5、5:4、21:9）
- 多任务并发生成

### 参考图模式
- 拖拽、点击上传或 Ctrl+V 快捷粘贴
- 支持多张参考图，左上角序号标记
- 图生图模式（使用 nano-banana-2 模型）

### 图库管理
- IndexedDB 本地持久化存储
- 合集分类管理
- 多选模式：批量选择、批量加入合集、批量下载
- 放大查看（滚轮缩放、拖拽平移、双击切换）
- 单张下载、批量下载为 ZIP 包
- 一键清空历史

### 用户体验
- 生成状态实时反馈（占位图 → 加载中 → 完成/失败）
- 响应式设计，适配各种屏幕
- 可调节侧边栏宽度
- 内置提示词指南

## 快速开始

### 配置 API Key
1. 点击右上角「设置」按钮
2. 输入你的 API Key
3. 点击保存

### 生成图片
1. 输入提示词
2. （可选）添加参考图
3. 选择模型和生成数量
4. 点击「生成」按钮

### 快捷操作

| 操作 | 方式 |
|------|------|
| 粘贴图片 | Ctrl+V |
| 放大查看 | 点击放大按钮 / 双击图片 |
| 缩放图片 | 滚轮滚动 |
| 平移图片 | 拖拽（需放大后） |
| 关闭查看 | ESC / 点击空白处 |
| 下载单张 | 点击下载按钮 |
| 批量下载 | 历史区「下载」按钮 |
| 复用设置 | 点击复用按钮 |
| 多选管理 | 点击「管理」按钮 |

## 技术栈

- **前端**：HTML5、CSS3、JavaScript（单文件应用）
- **存储**：IndexedDB
- **AI 接口**：Gemini API (newapi.asia)
- **压缩**：JSZip

## 提示词技巧

**文生图推荐结构**：
```
主体（数量/特征）+ 动作 + 场景（地点/时间/天气）+ 风格/媒介 + 构图/相机 + 光影/色彩 + 关键细节 + 约束条件
```

**图生图编辑时**：
```
保持（不可改变）+ 改变（修改什么）+ 如何（风格/强度/方向）+ 约束条件
```
